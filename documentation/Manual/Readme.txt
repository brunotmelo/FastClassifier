FastClassifier

Esta ferramenata foi criada com o intuito de ser rápida e flexível. Nesta ferramenta, é possível escolher que fases da classificação o desenvolvedor precisa para sua aplicação. Ele pode escolher fazer a classificação completa, ou pode apenas retirar as features e fazer a classificação fora da ferramenta. Esta possibilidade de escolher as funções, faz a classificação ser rápida, pois só serão efetuadas as operações necessárias.

Módulos
	O conjunto de comandos disponíveis se encontra na classe fastclassifier.Controller. Esta classe contém todos os módulos da aplicação.

Classificando
	A classe fastclassifier.FastClassifier, vem com esqueletos de comandos para fazer classificações. Montando as diversas funções do Controller para classificar áudios.

Setando parametros
	Na versão atual. Todos os parâmetros devem ser setados na classe fastclassifier.options.Parameters. Esta classe foi criada para juntar todas as escolhas de parâmetros em um só lugar visando tornar mais simples o uso da ferramenta.
